# Real Rain Credits

## Code and package

- Real Rain rebuild and expansion: Squishy1870.
- Built as a standalone Real Rain package from the supplied Dynamic Rain reference package.

## Thunder sound pass

- Thunder/storm source reference: Mixkit Thunder and Storm sound effects catalog.
- Mixkit lists free thunder sound effects for use under the Mixkit License, and the Mixkit license page lists Sound Effects under the Free License.
- This package uses curated OGG thunder layers arranged as distant rumbles, close cracks, rolling tails, and monsoon impacts.
- Processing applied for game use: OGG conversion/normalisation, low-pass distant rumble treatment, echo tails, and close-strike compression.

Source pages checked while preparing this pass:
